Ringtone Generator implemented on arduino

Authors: Shivdutt Sharma, Aditya Anand

This folder contains 4 files as follows:

1. generate_alphabet_mappings.py --> This is a python script to generate mappings from alphabets to frequencies.
2. alphabet_mappings.h --> This is a header file which contains matrices of mappings.
3. tone_generator.ino --> This is arduino file containing code to generate ringtone on arduino uno board which takes input as name.
4. Readme.txt
